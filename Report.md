# Отчёт о тестировании Money transfer

## Краткое описание

20.06.2021 было проведено функциональное тестирование приложения Money transfer.

На тестирование затрачено: 1 час

В результате тестирования выявлены следующие дефекты:
* [Значение дополнительного бонуса округляется с ошибкой](https://github.com/krisstallino/prgrmng_java_precision/issues/1#issue-925575190)

## Описание процесса тестирования

В процессе тестирования использовались следующие артефакты*:
* [Условия задачи №2](https://github.com/netology-code/javaqa-homeworks/tree/master/programming#задача-2---precision)

В качестве тестовых данных использовались данные указанные в условии задачи:
обычный бонус - 0.3
дополнительный бонус - 0.6
Ожидаемый результат - 0.9

Тестирование производилось в следующем окружении:
* macOS Big Sur ver.11.4
* Java ver.11.0.10
* IntelliJ IDEA CE 2021.1.2
